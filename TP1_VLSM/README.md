# TP1 — VLSM addressing and router configuration

**Objective:** Design a variable-length subnet mask (VLSM) addressing scheme for the given topology and configure router interfaces accordingly.  
**Network block used:** `192.168.33.128/25`

This folder includes:
- `vlsm_calculations.md` — full step-by-step VLSM calculations and table.
- `interfaces_table.md` — final address table for device interfaces.
- `configs/BR1_config.txt` and `configs/BR2_config.txt` — router configuration commands
- `notes.md` — short verification and troubleshooting checklist.
