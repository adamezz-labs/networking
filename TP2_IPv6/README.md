# TP2 — IPv6 addressing and device configuration

**Objective:** Configure IPv6 global addresses, link-local addresses, enable IPv6 routing, and verify end-to-end connectivity.

This folder includes:
- `ipv6_configuration_steps.md` — step-by-step configuration and explanation.
- `configs/R1_config.txt` — router commands (Cisco IOS style) for R1.
- `configs/S1_config.txt` — switch commands for SVI and SDM preference.
- `pcs_example.md` — example static IPv6 addresses for PC-A and PC-B.
